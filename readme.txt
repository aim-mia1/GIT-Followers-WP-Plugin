GIT Followers count and list with shortcodes.

== Description ==

GIT Followers WP Plugins provides ability to show GIT Follows count and list with Shortcodes anywhere in the site.

== Installation ==

Upload the GIT Followers WP Plugin to your blog, Activate it, and Update the user settings.

1, 2, 3: You're done!
